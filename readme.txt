== Description ==

The plugin automatically creates a new category for the new users in the categories specified by the administrator. Publish only the author can write in their categories.

== Installation ==

1. Upload `personal-authors-category` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
